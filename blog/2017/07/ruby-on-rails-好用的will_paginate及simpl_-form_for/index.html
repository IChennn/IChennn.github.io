[Ruby on Rails] еҘҪз”Ёзҡ„will_paginateеҸҠsimpl_ form_for

Jul 22, 2017
LearningRailsGem

з¶Із«ҷеӮійҖҒй–Җпјҡ https://gentle-inlet-97986.herokuapp.com/

дё»йЎҢжҳҜдёҖеҖӢжӣёи©•з¶Із«ҷпјҢйңҖиҰҒзҷ»е…ҘжүҚиғҪзҷјиЎЁи©•и«–пјҢеҸҜд»ҘиҮӘиЎҢдёҠеӮіжӣёзҡ„ең–зүҮпјҢд№ҹеҸҜд»Ҙжҗңе°ӢзҸҫжңүзҡ„жӣёзұҚиіҮиЁҠгҖӮ

пјҲеҹәжң¬жһ¶ж§ӢжҳҜз…§и‘—Rails 101зҡ„ж•ҷеӯёеҒҡеҮәдҫҶзҡ„пјҢиҮӘе·ұеҸҰеӨ–еҶҚеҠ дәҶдёҖдәӣе°ҸеҠҹиғҪпјү

жӯӨзҜҮдё»иҰҒзҙҖйҢ„еҰӮдҪ•дҪҝз”Ёwill paginate еҸҠ simple form forйҖҷе…©еҖӢж–№дҫҝеҸҲеҘҪз”Ёзҡ„gemдҫҶдҪҝз¶Із«ҷи®Ҡеҫ—жӣҙзҫҺи§ҖгҖӮ

use simple_form_for

Simple_form_for жҳҜдёҖеҖӢеҘҪз”Ёзҡ„gemпјҢеҸҜд»ҘиҮӘеӢ•иӘҝж•ҙж јејҸ

еҰӮжһңйӮ„жІ’е®үиЈқзҡ„и©ұпјҢиЁҳеҫ—е…ҲеңЁgemfileиЈЎйқўж–°еўһgem "simple_form" пјҢз„¶еҫҢ bundle install

дёӢдёҖжӯҘпјҡ

rails generate simple_form:install --bootstrap еҹ·иЎҢboostrapзҡ„иЁӯе®ҡ

е°ұеҸҜд»ҘдҪҝз”Ёsimple_form_forдәҶпјҒ

дҪҝз”Ёж–№жі•д№ҹеҫҲз°Ўе–®пјҢеҸӘиҰҒе°ҮеҺҹжң¬жҳҜform_for зҡ„йғЁеҲҶйғҪжӣҝжҸӣжҲҗsimple_form_for дёҰз…§и‘—иҰҸе®ҡзҡ„ж јејҸжӣҙж”№е…§е®№е°ұиЎҢдәҶгҖӮ

<%= simple_form_for @group do |f| %>
	<%= f.input :description, input_html: { class: "form-horizontal", :rows => 4} %>
<% end %>

:rows еҸҜд»Ҙз”ЁдҫҶиӘҝијёе…ҘжЎҶзҡ„й«ҳеәҰ

йҖҷжҳҜдҪҝз”Ёsimple_form_forзҡ„ж•Ҳжһңпјҡ

Simple_form_forжңғиҮӘеӢ•еҒөжё¬дёҚеҸҜз©әзҷҪзҡ„ж¬„дҪҚпјҢдёҰзөҰеҮәиӯҰе‘Ҡпјҡ

  • Reference : use form_for

    <%= form_for @group do |f| %>
    <%= f.label "description", :class => "string optional control-label" %>
    <%= f.text_area :description, :class => "string optional form-control" %>
 <% end %>

йҖҷж®өе’ҢдёҠйқўзҡ„simple_form_forиғҪйҒ”еҲ°зҡ„ж•Ҳжһңе№ҫд№ҺжҳҜдёҖжЁЎдёҖжЁЈзҡ„пјҢдҪҶжҳҺйЎҜе°ұеӣүе”ҶдәҶй»һпјёпјӨ

will paginate

дёҚз®ЎжҳҜд»Җйәјз¶Із«ҷпјҢзҙҖйҢ„дёҖеӨҡпјҢе°ұжңғиҗҪиҗҪй•·и®“дәәзңјиҠұж’©дәӮпјҢйҖҷжҷӮеҖҷжңүеҖӢеҘҪзңӢз°Ўе–®зҡ„еҲҶй ҒеҠҹиғҪе°ұйЎҜеҫ—ж јеӨ–йҮҚиҰҒдәҶпјҒ

е®үиЈқе®Ңwill_paginateеҫҢпјҢд№ҹиҰҒиЁҳеҫ—еңЁеҜ«йҖІGemfile иЈЎйқў

йҰ–е…ҲпјҢеңЁApp/controllers/groups_controller.rb з•¶дёӯеҠ е…Ҙ

@posts = @group.posts.recent.paginate(:page => params[:page], :per_page => 10)

жӯӨжҷӮиӢҘжҳҜзӣҙжҺҘrails server зҡ„и©ұжңғдёҖзӣҙеҮәзҸҫundefined method “recent”

йҖҷжҳҜеӣ зӮәзі»зөұз•¶дёӯдёҰжІ’жңүеҗҚзӮәrecentзҡ„ж–№жі•пјҢиҰҒиҮӘе·ұеҺ»app/models/post.rb дёӯеҠ е…Ҙ

scope :recent, -> {order("created_at DESC")}

жӯӨжҷӮжҮүи©Іе°ұеҸҜд»Ҙй ҶеҲ©зңӢеҲ°еҲҶй ҒдәҶпјҒ
